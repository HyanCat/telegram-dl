
## telegram-dl

### Usage

1. Create `config.json` file, like:

   ```json
    {
        "api_id": 123456,
        "api_hash": "xxxxx",
        "channels": [
            "channel_1",
            "channel_2",
            "channel_3"
        ]
    }
   ```

2. Run download script, command:

   ```bash
   python3 download_pic.py
   ```
